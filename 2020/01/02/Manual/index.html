实用手册

非详细教程，只记录重点

一. Java

  1. 下载
    清华大学开源软件镜像站-AdoptOpenJDK

    华为repo

    https://www.injdk.cn/

  2. 环境变量

  • Windows

    1. JAVA_HOME: path (Like C:\Program Files\Java\jdk1.8.0_251)
    2. PATH : %JAVA_HOME%\bin
    3. CLASS PATH: .;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar

  • Linux

    1. 解压/安装等

    2. vi /etc/profile

    4. . /etc/profile or source /etc/profile

二. Apache

1. Maven

  • 环境变量(Windows)

M2_HOME and MAVEN_HOME (Like D:\Program Files\Maven\apache-maven-3.6.3)

Path %M2_HOME\bin MAVEN_HOME\bin

  • 设置

2. Tomcat

1.环境变量

（1）变量名：TOMCAT_HOME 变量值：D:\Program Files\Apache Software Foundation\Tomcat\Tomcat 9.0\apache-tomcat-9.0.34
（2）变量名：CATALINA_BASE 变量值：%TOMCAT_HOME%
（3）变量名：CATALINA_HOME 变量值：%TOMCAT_HOME%
（4）Path中添加变量值：%CATALINA_HOME%\lib
（5）Path中添加变量值：%CATALINA_HOME%\bin

2.服务

cd to D:\Program Files\Apache Software Foundation\Tomcat\Tomcat 9.0\apache-tomcat-9.0.34\bin

三.MySQL

安装

  1. cd to mysql bin (Like D:\Develop\MySQL\mysql-8.0.20\bin)
  2. 安装服务 mysqld --install
  3. 初始化 mysqld --initialize --console (记录密码) (Like AVqaphEYh2%n)
  4. net start mysql
  5. mysql -uroot -p + password
  6. ALTER user 'root'@'localhost' IDENTIFIED BY '123456'
  7. my.ini

允许远程链接
